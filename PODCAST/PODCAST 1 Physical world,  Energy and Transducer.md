## ตอนที่ 1 — “โลกกายภาพ พลังงาน และภาษาที่ต้องแปล”

### **Host:**

สวัสดีทุกคนครับ ยินดีต้อนรับเข้าสู่รายการ “สนทนาภาษาเซนเซอร์” วันนี้เราจะเริ่มจากคำถามใหญ่ที่สุดในโลกของเซนเซอร์…

ก่อนจะมีคำว่า IoT ก่อนจะมีคำว่า ADC ก่อนจะมีคำว่าเซนเซอร์ เราต้องเข้าใจก่อนว่า **โลกกายภาพมันทำงานยังไง**

โลกนี้เต็มไปด้วย “พลังงาน” หลายรูปแบบมากนะครับ และพลังงานเหล่านี้มัน **แปลงหากันได้** เหมือนเป็นภาษาต่าง ๆ ที่พูดกันคนละแบบ แต่แปลกันได้เสมอ

### **Guest:**

ใช่ครับ และนี่แหละคือแก่นของคำว่า **Transducer** แต่ก่อนจะไปถึงทรานสดิวเซอร์ ผมอยากให้เรามองภาพใหญ่ก่อนว่า พลังงานในโลกนี้มีอะไรบ้าง และมันแปลงกันยังไง

แต่ก่อนจะไล่พลังงานทีละแบบ ผมอยากโยงให้ทุกคนจำง่าย ๆ ด้วยอุปมา…

# อุปมา: พลังงาน = ทวีป

### **Guest:**

ลองนึกภาพว่าโลกนี้มีหลายทวีป เช่น เอเชีย ยุโรป แอฟริกา อเมริกา ฯลฯ แต่ละทวีปก็มี “ภาษา” ของตัวเอง คนเอเชียพูดแบบหนึ่ง คนยุโรปพูดอีกแบบหนึ่ง ถ้าคนจากสองทวีปอยากคุยกัน ก็ต้องมี **ล่ามแปลภาษา**

พลังงานก็เหมือนกันครับ มันมี “ทวีปของตัวเอง” เช่น

- ทวีปพลังงานกล
- ทวีปพลังงานความร้อน
- ทวีปพลังงานแสง
- ทวีปพลังงานเสียง
- ทวีปพลังงานแม่เหล็กไฟฟ้า
- ทวีปพลังงานเคมี
- ทวีปพลังงานไฟฟ้า

แต่ละทวีปก็มี “ภาษา” ของตัวเอง เช่น แรง, การหมุน, อุณหภูมิ, ความสว่าง, การสั่น, สนามแม่เหล็ก ฯลฯ

### **Host:**

งั้นถ้าพลังงานแต่ละทวีปพูดกันคนละภาษา… เวลาเราจะเอาพลังงานจากทวีปหนึ่งไปใช้อีกทวีปหนึ่ง เราต้องมี “ล่าม” ใช่ไหมครับ?

และหากไม่อยากใช้ล่ามเราก็ต้องใช้ “ภาษากลาง” คือภาษาอังกฤษ  ใช่ไหมครับ? 

คนจีน คนไทย คนฝรั่ง ถ้าจะคุยกัน ตัวต่อตัว ก็ใช้ภาษาอังกฤษเป็นตัวกลาง

งั้นในโลกของพลังงาน… มี “พลังงานกลาง” แบบนั้นไหมครับ?

### **Guest:**

ถูกต้องแล้วครับ การที่พลังงานจะเปลี่ยนรูปได้มันต้องมีวิธีการหรือตัวการที่แปลงมันครับ

คุณเปรียบเทียบได้ดีมาก เพราะมันตรงเป๊ะกับสิ่งที่เกิดขึ้นในโลกจริง

และ "ภาษากลาง" ของพลังงาน ก็คือ **พลังงานไฟฟ้า** 

ลองคิดแบบนี้นะครับ
- พลังงานกล → พูดภาษาการหมุน
- พลังงานความร้อน → พูดภาษาอุณหภูมิ
- พลังงานแสง → พูดภาษาความสว่าง
- พลังงานเสียง → พูดภาษาการสั่น
- พลังงานแม่เหล็ก → พูดภาษาฟลักซ์
- พลังงานเคมี → พูดภาษาปฏิกิริยา

แต่ถ้าพลังงานเหล่านี้อยาก “คุยกันรู้เรื่อง” มันต้องแปลเป็น **พลังงานไฟฟ้า** ก่อน

เพราะไฟฟ้าเป็น “ภาษากลางของพลังงาน”

และล่ามในโลกของพลังงานเรียกว่า **Transducer** คืออุปกรณ์ที่ “แปลภาษา” ของพลังงานหนึ่งให้เป็นอีกพลังงานหนึ่ง


### **Host:**

คุณสามารถยืนยัน หรือยกตัวอย่างหน่อยได้ไหมครับว่า มันแปลงแบบตรงๆ หรือแปลงย้อนกลับไปมาได้ไหม

### **Guest:**

ได้ครับ ลองนึกภาพตามนี้นะครับ ถ้าเราอยากส่งความร้อนในไปตามบ้านเรือนในฤดูหนาวเราจะทำอย่างไร

แทนที่เราจะส่งกาซหุงต้มทีละถัง ซึ่งให้พลังงานความร้อนในเตาผิงแบบตรงไปตรงมา เราก็ส่งเป็นพลังงานไฟฟ้าไปแทน

เพราะว่านอกจากเอาไปทำความร้อน ยังทำอย่างอื่นได้อีกมาก

- พลังงานจากฟอสซิล  (ก๊าซ/น้ำมันดิบ)→ ความร้อน
- ความร้อน → ไอน้ำ
- ไอน้ำ → หมุนกังหัน
- กังหัน → ไฟฟ้า
- ไฟฟ้า → ฮีตเตอร์ (ความร้อน)

นี่คือการแปลภาษาพลังงานหลายทอดมาก แต่ที่สำคัญที่สุดคือ **การแปลงทุกขั้นตอนต้องเคารพ กฎอนุรักษ์พลังงาน**

> **พลังงานอาจสูญเสีย แต่ต้องไม่สูญหาย**

# พลังงานทุกชนิดสามารถแปลงหาพลังงานไฟฟ้าได้ (ทั้งโดยตรงและโดยอ้อม)

### **Host:**

งั้นเรามาไล่ดูทีละทวีปเลยครับ เริ่มจากทวีปพลังงานกลก่อน

### **Guest:**

**พลังงานกล (Mechanical Energy)** → ไฟฟ้า 

ตัวอย่าง
- กังหันลม  แปลงจาก **การหมุน** → **ไฟฟ้า**
- ไดนาโมจักรยาน แปลงจาก **การหมุน** → **ไฟฟ้า**
- POT แปลงจาก **การหมุน** → **แรงดันไฟฟ้า**

POT นี่แหละคือทรานสดิวเซอร์ที่จะได้ใช้ในแล็บในวันนี้ 
มันแปลง “ภาษาแห่งการหมุน” ให้เป็น “ภาษาไฟฟ้า”

### **Host:**

แล้วทวีปพลังงานความร้อนล่ะครับ?

### **Guest:**

**พลังงานความร้อน (Thermal Energy)** → ไฟฟ้า 

ตัวอย่าง
- เทอร์โมคัปเปิล แปลจาก **ความร้อน** → **แรงดันไฟฟ้า**
- NTC แปลจาก **อุณหภูมิ** → **ความต้านทานไฟฟ้า**

### **Host:**

ทวีปพลังงานแสง?

### **Guest:**

**พลังงานแสง (Light Energy)** → ไฟฟ้า 

ตัวอย่าง
- โซลาร์เซลล์ แปลจาก  **แสง** → **กระแสไฟฟ้า**
- LDR  แปลจาก **แสง** → **ความต้านทานไฟฟ้า**

### **Host:**

ทวีปพลังงานเสียง?
### **Guest:**

**เสียง (Sound Energy)** → ไฟฟ้า 

ตัวอย่าง
- ไมโครโฟน แปลจาก  **การสั่น** → **แรงดันไฟฟ้า**

### **Host:**

ทวีปพลังงานแม่เหล็กไฟฟ้า?

### **Guest:**

**สนามแม่เหล็ก → ไฟฟ้า** 

ตัวอย่าง
- Hall Effect Sensor แปลจาก  **สนามแม่เหล็ก** → **แรงดันไฟฟ้า**

#  ทำไมไฟฟ้าถึงเป็น “ทวีปศูนย์กลาง”?

### **Host:**

งั้นคำถามใหญ่เลยครับ ทำไมไฟฟ้าถึงเป็นพลังงานที่สำคัญที่สุดในโลกเทคโนโลยี?

### **Guest:**

เพราะไฟฟ้าเป็นพลังงานที่

- ส่งผ่านได้ง่าย
- แปลงเป็นข้อมูลได้ง่าย
- ประมวลผลได้ด้วยคอมพิวเตอร์
- แปลงกลับเป็นพลังงานรูปอื่นได้
- ควบคุมได้แม่นยำ
- เก็บในแบตเตอรี่ได้
- ตรวจวัดได้ละเอียดมาก

พูดง่าย ๆ คือ **ไฟฟ้าคือภาษากลางของเทคโนโลยี**

เดี๋ยวผลสรุปตัวอย่างให้ในตารางด้านล่างนี้นะครับ

| ทวีป (โดเมน) พลังงาน | ภาษา (รูปแบบพลังงาน) | ตัวอย่าง transducer |
| -------------------- | -------------------- | ------------------- |
| กล                   | การหมุน / แรง        | POT, กังหัน         |
| ความร้อน             | อุณหภูมิ             | Thermocouple, NTC   |
| แสง                  | ความสว่าง            | LDR, Solar Cell     |
| เสียง                | การสั่น              | Microphone          |
| แม่เหล็ก             | ฟลักซ์แม่เหล็ก       | Hall Sensor         |
| เคมี                 | ปฏิกิริยา, รส, กลิ่น | Battery             |

# จากไฟฟ้า → ข้อมูล → คอมพิวเตอร์ → IoT

### **Host:**

งั้นพอแปลงทุกอย่างเป็นไฟฟ้าได้ เราก็แปลงมันเป็นข้อมูลได้?

### **Guest:**

ใช่ครับ สมัยก่อน เราก็สร้างทรานซดิวเซอร์กันมากมาย เพื่อแปลงพลังงานในรูปแบบต่างๆ เป็นพลังงานไฟฟ้า แล้วส่งให้คอมพิวเตอร์ประมวลผล ตามลำดับนี้เลยครับ

1. โลกกายภาพ → พลังงาน
2. พลังงาน → ไฟฟ้า (Transducer)
3. ไฟฟ้า → ข้อมูล (ADC)
4. ข้อมูล → การประมวลผล (คอมพิวเตอร์)
5. คอมพิวเตอร์ → การตัดสินใจ
6. การตัดสินใจ → การควบคุมอุปกรณ์

### **Host:**

ครับ นั่นคือกระบวนการควบคุมอัตโนมัติสมัยใหม่ ที่นักศึกษาวิศวกรรม หรือระบบควบคุมต่างๆ ต้องเรียน

### **Guest:**

ถูกต้องแล้วครับ แต่ปัจจุบัน เมื่อไมโครคอนโทรลเลอร์มันเล็กลง อุปกรณ์อิเล็กทรอนิกส์ต่างๆ ก็เล็กลง
เราก็ไม่ต้องเดินสายจากเซนเซอร์มาหาคอมพิวเตอร์แล้ว

เราจะจับคอมพิวเตอร์ ไปอยู่บ้านเดียวกับเซนเซอร์ตัวเล็กจิ๋วนั่นซะเลย และก็ต่ออินเทอร์เน็ตให้มัน

### **Host:**

อ๋อ... นี่ก็คือที่มาของ Internet of Things 
### **Guest:**
ใช่ครับ พอเชื่อมอินเทอร์เน็ตได้ อุปกรณ์ที่มีเซนเซอร์ก็กลายเป็น IoT
